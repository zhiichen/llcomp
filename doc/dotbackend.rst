Dot Backend
*****************
.. module:: DotBackend 
   :synopsis: Converts IR to Dot


The :mod:`DotBackend` module contains a :class:`DotWriter`, capable of 
produce Dot language code from the IR. 

This backend is used in the :mod:`Tools.Debug` tool.

The :class:`DotWriter` implements a visitor pattern.
