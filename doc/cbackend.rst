CBackend
*****************
.. module:: CBackend
   :synopsis: Converts IR to C with OpenMP extensions


The :mod:`CBackend` module contains two writers, :class:`CWriter` and :class:`OmpWriter`, 
which produces **C** and **OpenMP** code

.. note::
   This backend is widely used on llCoMP.
